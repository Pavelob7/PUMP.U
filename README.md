# Как задеплоить сие чудо?

1. Скачать всё с данной ветки (readme можно удалить)
2. Засунуть на сервак
3. Прописать:
    ```
   npm install --production
   npm start
   ```
4. По умолчанию запустится приложение по адресу localhost:3000
5. Зайти в папку nginx\conf и в файле nginx.conf в разделе  location / { прописать:
   ```
    proxy_pass http://localhost:3000;  # Порт, на котором запущен Next.js сервер
   ```
6. Прописать:
   ```
   start nginx
   ```
